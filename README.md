# 🌤️ Roman Urdu AI Weather Assistant Agent

An intelligent, conversational AI Agent built using **n8n** that checks the live weather of any city and provides friendly, local recommendations in Roman Urdu.

## 🚀 Features
- **Live Weather Updates:** Integrated with OpenWeatherMap API.
- **Friendly AI Personality:** Talks in friendly Roman Urdu.
- **LLM Powered:** Uses Meta Llama 3.1 (via OpenRouter) for smart reasoning.
- **Fully Local & Free:** Built using 100% free-tier APIs.

## 🛠️ How to Use
1. Import the `weather-ai-agent.json` file into your **n8n** instance.
2. Add your **OpenRouter** and **OpenWeatherMap** API keys in the credentials.
3. Start the chat trigger and enjoy!
